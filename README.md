# Mini-Vault

Desktop note-taking application that visualizes notes as interactive 3D spheres in a spatial canvas, connected by semantic similarity.

## Overview

Desktop note-taking application that visualizes notes as interactive 3D spheres in a spatial canvas, connected by semantic similarity.

## Features

- 3D graph visualization with force-directed layout
- Automatic semantic connections via Jaccard similarity
- Rich note editor with text, bullets, and checklists
- Workspaces for organizing notes into color-coded groups
- Search and timeline filtering
- Spaced repetition review system with exponential intervals
- Backlink detection for cross-referencing
- Export notes as JSON with connection data

## Installation

### Prerequisites

- TypeScript (React 18, Three.js, Electron)
- Git

### Steps

```
git clone https://github.com/judeako402-dotcom/Mini-Vault.git
cd Mini-Vault
npm install
npm run dev
```
